## MCP Marketplace plugin

To enable the repository to use the modernize-dotnet plugin in a Copilot/MCP session, run:

/plugin marketplace add dotnet/modernize-dotnet

A repository manifest was added at `.mcp/plugins.json` listing the plugin for convenience.
